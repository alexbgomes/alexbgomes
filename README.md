# AlexBGomes.dev

A modern, interactive portfolio website built with React and TypeScript.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

-   [Node.js](https://nodejs.org/) (v18 or higher)
-   [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/alexbgomes/alexbgomes.git
    cd alexbgomes
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Development

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Build

To build the project for production:

```bash
npm run build
```

## Deployment

Hosted through GitHub Pages, served through personal domain alexbgomes.com

To deploy to GitHub Pages (this will also build before deploying):

```bash
npm run deploy
```
